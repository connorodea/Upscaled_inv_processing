Upscaled Inventory Processing - Quick Start (macOS)

1) Install
- Download Upscaled_inv_processing_mac.zip from the email.
- Double-click to unzip.
- Open the extracted folder.
- Double-click install.command.
  - If macOS blocks it: right-click > Open > Open.
  - If asked for your password, enter it to allow install.

2) Open the app
- Double-click the desktop icon: Upscaled Inventory
  OR
- Open Terminal and type: upscaled

3) Printer setup (Dymo 450)
- In the menu, choose: Setup printer (auto-detect).
- It will pick the Dymo 450 and use 2x1 labels.

4) Normal use
- Enter PID and UPC (or Manufacturer + Model).
- Print SKU label when prompted.
- The master manifest row is marked Processed with a timestamp.

5) Google Sheets sync
- Happens automatically after every batch export.
- No login needed.

Troubleshooting
- If the app will not start, restart the computer and rerun install.command.
- If printing fails, rerun Setup printer (auto-detect).
