Upscaled SOP

Receiving + Processing Workflow

1) Receive Manifest

  1. Open the web app and go to Receive Manifest.
  2. Enter the order # and unit count.
  3. Generate and print PID-UID labels.
  4. Apply a PID-UID label to every unit as you unload.

2) Process Inventory

  1. Go to Process Inventory Item.
  2. Scan or type the PID-UID label.
  3. Select grade.
  4. Enter UPC or Manufacturer + Model.
  5. Assign Warehouse Tag.
  6. Print the SKU label and apply it to the unit.

3) Build Inventory Hub

  1. Click Build Inventory Hub when you want fresh CSVs.
  2. Download processed/unprocessed CSVs for analysis.

4) Export Batches

  1. Use Batch Tools to export a batch.
  2. Confirm the batch file is saved.
Tip: If anything fails, retry the step or notify a lead. The CLI can still perform every action.
